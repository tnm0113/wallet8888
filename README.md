### HOW TO RUN

1.  Install nvm, nodejs
2.  Clone project

```bash
git clone https://github.com/tnm0113/wallet8888.git
```

3.  Install ts-node:

```bash
npm install -g ts-node
npm install
```

4.  Copy file .env and edit option you want

```bash
cp .env.default .env
```

5.  Start:

```bash
npm start

```
